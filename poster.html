Title : Virtual F1 Car Designer — Real-Time Component Integration & Performance Simulation

Abstract

The Virtual F1 Car Designer is a web-based application that enables users to build custom Formula 1 cars by selecting authentic components from four real F1 teams — Ferrari, Mercedes, Red Bull, and McLaren. The system performs intelligent compatibility checking between cross-team components, runs physics-based performance simulations across 12 real FIA circuits, and integrates real-world telemetry data via the FastF1 Python library. Users can visualize live track animations, compare simulated performance against actual F1 race data, and save configurations for iterative design optimization.

Introduction

Formula 1 engineering involves complex interactions between powertrain, aerodynamics, tires, and braking systems. Understanding how these components interact requires access to expensive simulation tools typically unavailable to enthusiasts and students.


This project democratizes F1 car design by providing an interactive platform where users can experiment with component combinations, understand compatibility constraints, and observe realistic performance outcomes. The application bridges the gap between real F1 data and educational simulation, making motorsport engineering accessible through modern web technologies.

Methodology

  • Frontend: React 18 with Vite build system, Tailwind CSS 4 for styling, Recharts for data visualization
  • Backend: Python FastAPI server wrapping the FastF1 library for real telemetry data access
  • Component Selection: Users choose Engine, Tires & Brakes from 4 F1 teams with real-world specifications
  • Compatibility Engine: Rule-based system checking cross-team component conflicts (thermal, electrical, mechanical)
  • Simulation Engine: Physics model incorporating tire degradation curves, fuel load effects, weather impact, and circuit-specific characteristics
  • Real Data Integration: FastF1 API provides actual lap times, speed traces, driver telemetry, and race results
  • Track Animation: Catmull-Rom spline interpolation for smooth car movement along 12 calibrated circuit paths
  • Data Persistence: LocalStorage for saved configurations and version history tracking

Result

  • Simulates F1 car performance across 12 real circuits with lap times within 2–5% of actual F1 data
  • Real-time track animation with car dot tracing circuit paths at 1x–10x speed
  • Live telemetry visualization: speed, throttle, brake & gear data from actual F1 sessions
  • Driver comparison overlay charts for side-by-side analysis
  • Compatibility system detects 5 cross-team conflict types with severity levels
  • Sub-second simulation response time for 5–78 lap races

Conclusion

The Virtual F1 Car Designer successfully demonstrates that complex motorsport engineering concepts can be made accessible through well-designed web applications. By combining simulated physics models with real-world FastF1 telemetry data, the platform provides educational value while maintaining engineering accuracy. The modular architecture allows easy extension with additional teams, circuits, and simulation parameters. Future work includes aerodynamic modeling, pit stop strategy optimization, and multi-car race simulation.
